# ADR-004: بلا Inngest — Cron مؤمّن بسرّ

**الحالة**: مُعتمد

## القرار
لا منصّة مهام خلفية (Inngest/Queue). دورة حياة الاشتراكات/التذكيرات عبر مسار `/api/cron/subscription-reminders` مؤمّن بـ `CRON_SECRET`، يُشغَّل بـ Vercel Cron (ويُحفَّز أيضًا عند فتح تبويب الفوترة).

## البدائل المرفوضة
- Inngest/طابور مخصّص: تعقيد بلا حاجة — المهام قليلة ودورية.

## النتائج
- (+) بساطة، بلا اعتمادية خارجية.
- (−) لا إعادة محاولة/طابور متقدّم — كافٍ للحمل الحالي. مراجعة عند تعقّد المهام.
