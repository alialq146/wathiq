# ADR-005: بلا تخزين كائنات (R2/S3)

**الحالة**: مُعتمد

## القرار
لا تخزين كائنات. رفع الـ PDF يُعالَج في الذاكرة كـ base64 (بحدّ ~4.4MB) ولا يُخزَّن. التصدير يُولَّد عند الطلب ويُرسَل مباشرةً.

## البدائل المرفوضة
- R2/S3 لتخزين المستندات/المخرجات: بنية وتكلفة بلا حاجة — لا ميزة تتطلّب استمرار الملفات بعد.

## النتائج
- (+) لا إدارة أصول ولا أذونات تخزين.
- (−) لا استمرار للملفات المرفوعة/المصدَّرة — مراجعة عند ميزة تتطلّب ذلك.
