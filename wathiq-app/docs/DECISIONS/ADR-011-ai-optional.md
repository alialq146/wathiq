# ADR-011: الذكاء الاصطناعي اختياري

**الحالة**: مُعتمد

## القرار
الذكاء الاصطناعي ميزة اختيارية بالكامل: لا يعمل إلا بضغطة مستخدم صريحة، وبلا `ANTHROPIC_API_KEY` يعمل التطبيق كاملًا يدويًا (مشاريع/متطلبات/تصدير) على بيانات احتياطية.

## النتائج
- (+) المنتج قيّم حتى بلا ذكاء؛ تدهور رشيق.
- (+) حدّ خصوصية: بيانات المستخدم تُرسَل للتحليل فقط عند طلبه الصريح.
- (−) لا failover لمزوّد بديل — التعطيل صلب.
