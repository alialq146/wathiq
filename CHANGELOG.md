## [0.6.0] — 2026-07-01 · سير الاعتماد والحالة
### أُضيف
- زرّا **«طلب الاعتماد»** و**«طلب معلومات إضافية»** في لوحة التفاصيل يغيّران حالة المتطلب فعليًا في القاعدة عبر Server Action، مع حفظ فوري وتحديث شارة الحالة في التفاصيل والقائمة.
- حالة تحميل (أيقونة دوّارة)، ورسائل نجاح/خطأ واضحة، وتعطيل ذكي للأزرار عند بلوغ الحالة نفسها أو عند عدم الاتصال بقاعدة البيانات.
- البحث والتصفية والترتيب في قائمة المتطلبات (بحث بالرقم/النص · فلترة بالحالة والأولوية · ترتيب) مع زر **«مسح كل الفلاتر»**.

## [0.5.0] — 2026-06-30 · الذكاء الاصطناعي
### أُضيف
- تحليل وثائق المتطلبات فعليًا عبر **Claude (claude-opus-4-8)** مع مخرجات منظّمة.
- نقطة `/api/analyze` (محميّة بمفتاح `ANTHROPIC_API_KEY`).
- زر **«حفظ في القاعدة»** يضيف المتطلبات المستخرجة إلى Neon.
- شاشة تحليل تفاعلية: لصق نص → تقدّم مباشر → نتائج حقيقية (ملخّص، خطوات، توصيات، ثقة).

## [0.4.0] — 2026-06-30 · لوحة التعديل
### أُضيف
- إضافة/تعديل/حذف المتطلبات من الواجهة عبر Server Actions تكتب في القاعدة.
- نافذة نموذج كاملة + أزرار تعديل/حذف على كل بطاقة.
### تغيّر
- سكربت البذرة صار **«مرة واحدة»** حتى لا تُمسح التعديلات عند إعادة النشر.

## [0.3.0] — 2026-06-30 · قاعدة البيانات
### أُضيف
- طبقة بيانات بـ **Prisma + Neon Postgres** مع **تراجع آمن** للبيانات الافتراضية.
- نقطة `/api/health` تكشف مصدر البيانات (`database` / `fallback`).
- نطاق مخصص **`wathiq-ai.vercel.app`**.
