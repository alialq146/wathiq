**Select** — styled native dropdown matching the input system.

```jsx
<Select label="الأولوية" options={["حرجة","عالية","متوسطة","منخفضة"]} />
<Select label="المُسند إليه" options={[{value:"sa",label:"سارة"},{value:"om",label:"عمر"}]} />
```

Accepts string options or `{value,label}`. Sizes `sm`/`md`/`lg`.
