**Button** — primary action control for Wathiq workspaces. Use for form submits, primary CTAs, and toolbar actions.

```jsx
<Button variant="primary" iconStart={<i data-lucide="sparkles" />}>تحليل بالذكاء الاصطناعي</Button>
<Button variant="secondary">حفظ كمسودة</Button>
<Button variant="ghost" size="sm">إلغاء</Button>
```

Variants: `primary` (trust blue), `secondary` (outlined), `ghost` (transparent), `brand` (navy), `danger` (red). Sizes: `sm` · `md` · `lg`. Supports `iconStart` / `iconEnd`, `fullWidth`, `disabled`.
