**Badge** — small pill for counts, labels, and generic statuses. For requirement lifecycle states prefer `StatusBadge`.

```jsx
<Badge tone="ai" dot>الذكاء الاصطناعي</Badge>
<Badge tone="success">معتمد</Badge>
<Badge tone="warning" variant="solid">3</Badge>
```

Tones: `neutral` `info` `success` `warning` `danger` `ai`. Variants: `soft` (default), `solid`. Optional leading `dot`.
