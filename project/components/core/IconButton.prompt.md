**IconButton** — compact icon-only control for toolbars, table rows, and panel headers.

```jsx
<IconButton label="مزيد من الخيارات"><i data-lucide="more-horizontal" /></IconButton>
<IconButton variant="outline" label="تصفية"><i data-lucide="filter" /></IconButton>
```

Variants: `ghost` · `soft` · `outline`. Sizes `sm`/`md`/`lg`. Always pass `label` for accessibility.
